HTTPS / SSL

xmstore unterstützt auf Anfrage die Verschlüsselung der Verbindung zwischen Browser und Server. Dadurch erscheinen auch nicht mehr die Sicherheitswarnungen im System ("unsichere Verbindung").

Dazu muss ein individuelles Zertifikat für Ihr System generieren und eingebunden werden. Das können wir pauschal für 49 Euro anbieten. Weitere dauerhaften Kosten dabei entstehen nicht.

Auf Wunsch können Sie auch ein eigenes Zertifikat bei einem Anbieter für Ihre Domain anfordern und uns die Daten zur Einbindung weiterleiten.

htaccess Weiterleitung HTTPS

Damit die bisherigen Zugriffe von HTTP:// auf HTTPS:// weitergeleitet werden, ist es notwenig eine neue Weiterleitungsregel angelegt werden. 

RewriteCond %{HTTPS} off
RewriteRule .* https://%{HTTP_HOST}%{REQUEST_URI} [NE,QSA,L,R=301]

Domain Konfiguration HTTPS

In der Domainkonfiguration müssen danach auch die betroffenen Domains auf HTTPS umgestellt werden. Dadurch werden in der XML-Sitemap aufgeführte Domains auf HTTPS umgestellt, und Besucher im Frontend werden automatisch von HTTP zu HTTPS weitergeleitet. Sonst verweist die Sitemap auf HTTP und die Weiterleitungsregel muss jeden Zugriff von Google erst noch selbst umleiten.

Die Umstellung erfolgt unter "Einstellungen > Domain > (Hauptdomain auswählen) > HTTPS" anhaken.

Responsive Design

Für Google ist mittlerweile ein wichtiges Kriterium für das Ranking, ob eine Website auf Mobilgeräten (Handy, Tablet) sowie in verschiedenen Auflösungen angezeigt und benutzt werden kann. Diese Seiten zu erstellen ist deutlich komplexer und nicht einfach. Beim Template "Serious" haben wir diese Leistung von einem externen Programmierer durchführen lassen (30 Stunden).

Prüfen kann man eigene Seiten mit dem Tool von Google https://search.google.com/test/mobile-friendly

Optimierte Templates

Das Template "Serious Responsive" unterstützt diese Darstellung und wird dadurch von Google bevorzugt gelistet. Ein weiterer Vorteil ist die dynamische Veränderung der Darstellung bei verschiedenen Auflösungen von mobilen Geräten.

Das Template ist derzeit in Testbetrieb und kann von jedem xmstore Kunden verwendet werden. Sie erhalten es derzeit auf Anfrage. Damit das Template genutzt werden kann, müssen Ihre individuellen Änderungen am Template übernommen werden (z.B. Logo und Farbanpassungen). Gerne unterstützen wir Sie mit unseren Leistungen zur Übertragung der Änderungen.

Sprachordner in der URL

Das Template "Serious Responsive" unterstützt weiterhin die von Google weiterhin empfohlene URL-Struktur. Hier werden die verschiedenen Sprachversionen seperat in Unterordner aufgerufen.

agentur.tld/de/ (Deutsche Fassung)
agentur.tld/en/ (Englische Fassung)
agentur.tld/es/ (Spanische Fassung)
agentur.tld/fr/ (Französische Fassung)
usw.


Keywordlinks der Detailseite:

Hier nuss noch in der htaccess-Datei eine Regel zur Weiterleitung der Keywordlinks der Detailseite eingefügt werden. 

RewriteRule ^(.+)/tag/(.*?)/?$ /$1/s/index0.html&q[]=$2 [QSA,L,R=301]

Generierung der Sitemap.xml

Innerhalb von xmstore wird die gesamte Seitenstuktur automatisch für die Indizierung von Google vorbereitet und als xml-Datei zur Verfügung gestellt.

Wichtig ist, dass Sie in xmstore dazu die richtige Domain für die Generierung der Sitemap auswählen. Das ist i.d.R. Ihre Hauptdomain (z.B. www.domain.tld). Falls dort noch eine xmstore Subdomain (z. B. agentur.xmstore.de) eingetragen ist, sollten Sie diese auf die Hauptdomain ändern.

Einstellungen > Domains > Sitemap "aktiv"


Doppelter Content

Verhindern Sie bei mehreren Domain mit Weiterleitungen in der htaccess, dass eine Seite unter mehreren Links aufrufbar ist.

Beispiel:

  1. www.domain.tld/de/image/beispielbild-18079.html
  2. http://www.domain.tld/de/image/beispielbild-18079.html
  3. https://www.domain.tld/de/image/beispielbild-18079.html
  4. https://www.weiteredomain.tld/de/image/beispielbild-18079.html


Hier würde Google die Kraft der Seite auf diese 4 Seiten verteilen. Sinnvoller wäre es, wenn es Weiterleitungen auf eine einzige Seite geben würde (in dem Fall wäre Nr. 3 ideal) und damit diese Seite die volle Kraft entwickelt.
