Aktionen & Navigation

In der grauen Leisten haben Sie verschiedene Funktionen zur Durchführung von Aktionen und zur Navigation durch die Daten.

Aktionen durchführen

  1. Wählen Sie dazu die Medien mit der Checkbox aus mit denen Sie eine Aktion durchführen möchten (Hintergrund verändert sich gelb).
  2. Wählen Sie im Pull-Down-Menu die gewünschte Aktion aus. Bei manchen Aktionen (z. B. Löschen von Daten) ist eine Bestätigung notwendig.

Seitennavigation

Um auf die nächste Seite zu wechseln, klicken Sie auf das Symbol vor > (> Dreieck nach rechts). Um auf die letzte Seite zu wechseln, klicken Sie auf das Symbol vor (>| Dreieck nach Rechts + Balken). Genauso können Sie die Seite auf die Sie welchseln möchten auch direkt in die Eingabebox eingeben.

Ergebnisse pro Seite

Die Anzahl der Ergebnisse pro Seite kann von Ihnen bis zu einem Wert 999 frei eingestellt werden.

Wichtig
Bei manchen Prozessen, wie z. B. im Multi-EditorMit dem Multi-Editor verschlagworten Sie eine beliebige Anzahl von Medieninhalte in einem Schritt., sollten Sie mit kleineren Werten arbeiten, um Wartenzeiten beim Öffnen und Speichern und die Gefahr eines Datenverlustes zu reduzieren. Auch kann es bei großen Mengen von Medien abhängig von der Serverkonfiguration zu Fehlermeldungen kommen.


Darstellung anpassen

Jeder Benutzer kann die Darstellung der Datenübersicht nach seiner Vorstellung anpassen. Dazu in der grauen Menuleiste auf Anpassen klicken und die Änderungen durchführen. Diese Änderungen werden pro Benutzer und pro Datenübersicht individuell im System gespeichert.


Anpassungsoptionen

Sie können entscheiden, welche Daten angezeigt werden (Checkbox), in welcher Reihenfolge die Daten erscheinen (Position) und die Spaltenbreite beeinflussen (Breite).

Wichtig
Neue Funktionen und Aktionen konnten in die KachelansichtIn diesen Ansichten werden Daten aus der Datenbank in der Form von Tabellen oder gekachelt dargestellt. nicht übernommen werden. Deswegen kann diese alternative Ansicht leider nicht mehr angeboten werden.





Filter

Für die gezielte Suche können in verschiedenen Eingabeboxen in der dunkelblauen Zeile Filter gesetzt werden. Bei einem gesetzten Filter färbt sich der Hintergrund gelb.

Sortierung der Daten

Um die Reihenfolge zu verändern, klicken Sie auf die kleinen Pfeile in der dunkelblauen Zeile.

CSV-Export

Die aktuell in der jeweiligen Datenübersicht angezeigten Daten können Sie über Kopieren und Einfügen in eine Tabellenkalkulation übertragen. Wählen Sie beim Import der Datei das Tennzeichen Komma und den Zeichensatz "Unicode UTF-8".

Einschränkung
Die Verwendung dieses Features mit Internet Explorer kann zu unerwartetem Verhalten und Fehlermeldungen führen.
Wichtig
Es werden nur die aktuell im Grid angezeigten Daten (max. 999 Ergebnisse pro Seite) exportiert.

Weiterhin werden verknüpfte Daten (z. B. Kundenname im Warkenkorb-Grid) und Sortierungen nicht übertragen.
