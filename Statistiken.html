Statistiken

xmstore bietet zahlreiche Möglichkeiten der Auswertungen in Form von 3 flexiblen Statistiken.

Jede Statistik kann durch Auswahlen und Filter individuell angepasst werden. Möchten Sie eine Statistik in einer bestimmten Form später wieder so aufrufen, können Sie die Einstellungen über ein Lesezeichen speichern. 

xmstore.de/admin/compbarstat?f[data]=compcustomer&f[date][start]=2012-09-05&f[date][end]=2013-02-22&f[mediatype]=0&f[user]=0&f[paymentmodule]=0&f[paymentstatus]=0

Zugriffsrechte:
Die jeweiligen Zugriffsrechte auf die Statistiken hängen davon ab, welche Rechte dem Benutzer bzw. der Gruppe zugewiesen wurden. Wenn ein Benutzer oder eine Gruppe nicht den Parameter "Alles sehen" gesetzt hat, kann er nur auf seine persönlichen Daten zugreifen.

Statistik Bestellungsprotokoll

Beispiele

  • Welche Kunden haben in einem bestimmten Zeitraum am häufigsten bestellt?
  • Wer sind die erfolgreichsten MedienautorenMedienautoren (Urheber) erstellen künstlerische Werke in immaterieller Form (z. B. Fotografien). in einem bestimmten Zeitraum?
  • Welche Medientypen wurden in einem bestimmten Zeitraum am häufigsten verkauft?
