Suche

Für die Suche können zahlreiche Filter verwendet werden, um eine Suche zu starten oder das Suchergebnis einzuschränken:

→ Hauptartikel: Suche


Medialist-Ansicht

Das Suchergebnis wird im Form von kleinen Vorschaubildern (Thumbnails) dargestellt. Möchte der Kunden mehr Informationen über einen Artikel sehen, kann er die Mouse-Over-Ansicht öffnen oder mit der Lupe zur Detail-Ansicht wechseln.

→ Hauptartikel: Medialist-Ansicht

Detail-Ansicht

Auf der Detailansicht sieht ein Kunde eine größere Ansicht des Artikel sowie die in der Datenbank dazu gespeicherten Informationen.

→ Hauptartikel: Detail-Ansicht

Warenkorb

Aus der Thumbnail-Ansicht sowie in der Detail-Ansicht ist es möglich, einen Artikel im Warenkorb für eine Bestellung vorzumerken. Diese Artikel wird dann im Mini-Warenkorb eingeblendet.

Klickt man dort auf dem Button "Warenkorb", kann man diesen bearbeiten. Möchte man einen Warenkorb wieder aufrufen ohne im System angemeldet zu sein, reicht es, sich die URL des Warenkorbs als Lesezeichen zu speichern (z. B. domain.tld/cart?cart=513e7dd67e4).

Möchten Sie einen Artikel mehrfach bestellen (z. B. zwei verschiedenen Verwendungen "Rights Managend"), klicken Sie auf "Dublizieren".

Sichern von Warenkörben im Kundenbereich

Sobald sich der Kunde im System angemeldet ist, werden die angelegten Warenkörbe im Kundenbereich gespeichert. Dort kann der Kunde beliebige Warenkörbe wieder aufgerufen, solange diese nicht bestellt oder gelöscht wurden.

Wichtig
Der Link "Kasse" erscheint erst, wenn bei allen Kriterien, die zur Preisberechnung notwendig sind, eine Auswahl getroffen wurde.


Anmeldung

Um einen Warenkorb einer Kundennummer zuzuordnen oder zu bestellen, muss man sich im System anmelden.


Registrierung

Ist der Kunde noch nicht im System registiert, muss er das Kundenformular einmalig ausfüllen.

Das Feld USt.-Id.-Nr erscheint nur, wenn im gewählen Land entsprechend konfiguriert wurde (Reverse-Charge-Verfahren).

→ Artikel: Rechnung#Länder

Einloggen

Hat man sich bereits erfolgreich registiert, kann man sich mit seiner Kundennummer oder E-Mail-Adresse im System anmelden.

Kundenbereich

Nachdem dem erfolgreichen Login ist der Kunde im Kundenbereich. Dort werden ihm die Kundendaten, die letzten Warenkörbe und die letzten Bestellungen angezeigt.


Bestellung

Ist der Kunde im System eingeloggt und sind bei allen Artikeln im Warenkorb alle Preisoptionen ausgewählt, erscheint der Link "Kasse" um den Bestellprozess einzuleiten.


Zahlung & Optionen

Auf dieser Seite entscheidet sich der Kunde für eine der Zahlungsmöglichkeiten, die der Betreiber aktiviert hat. Während des Zahlungsprozesses können einzelne Schritte über die Seiten eines externe Dienstleisters (z. B. PayPal) erfolgen.

Dieser Schritt wird übersprungen, falls alle zu bestellenden Artikel das Preismodell "Anstrich" gewählt haben oder alle zu bestellenden Artikel den Preis 0,00€ haben.

Bestellungsbestätigung

Auf dieser Seite werden alle Informationen über den Kaufvorgang zusammengefasst dargestellt. Anschließend kann der Kauf verbindlich abgeschlossen werden.

Stornierung von Bestellung

Erstellte Bestellungen und Gutschriften können nachträglich nicht mehr bearbeitet werden, da das intern zu technischen Problemen in der Abrechnung führen würde. In so einem Fall sollte man die Bestellung auf storniert setzen.

Download / Versand

Zum Abschluss erhält der Kunde Zugriff auf die Rechnung und die direkt über das System herunterladbaren Artikel. Diese Seite kann auch über den Kundenbereich später wieder aufgerufen werden.

→ Hauptartikel: Kundenbereich
