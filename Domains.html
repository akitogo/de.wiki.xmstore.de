Domain

Eine Domain (z. B. www.agentur.de) können Sie bei einem Anbieter Ihrer Wahl bestellen und für xmstore einsetzen. Sie benötigen dazu nur den Domainnamen und keinen Webspace (Speicherplatz auf einem Server)! Wenn Sie unsere Hosting-Leistungen nutzen, muss die Domain auf die IP 85.88.2.186 (oder eine andere von uns genannte IP) zeigen. Bei xmstore Basic ist nur die Nutzung der Subdomain (z.B. name.xmstore.de) möglich.

Zuweisung einer Domain zu einem Server

Um eine eigenen Domain (z. B. www.agentur.de) mit einem unserer Server zu verbinden, verwenden Sie bitte folgende IP-Adresse: 212.83.34.159

Dazu sind folgende Schritte notwendig

  • Melden Sie sich im System Ihres Domain-Provider ein
  • Wählen Sie die gewünschte Domain aus und bearbeiten Sie die Einstellung
  • Passen Sie die DNS-Einstellung des Servers an und tragen die neue IP des Servers als A-Record ein

Ab dem 08.08.2019 (Neues Rechenzentrum) muss die Konfiguration wie folgt lauten. Bitte nennen Sie diese neuen Daten Ihrem Domainprovider oder nehmen Sie diese Änderung selbst durch: 

domain.tld. IN    A       212.83.34.159 
domain.tld. IN    AAAA    2a00:f48:caff:924::1
*           IN    CNAME   domain.tld.

Viele Domain- und DNS-Anbieter bieten einfache Werkzeuge zum umstellen der IP. Manuelles bearbeiten der DNS-Zone ist oft nicht erforderlich. Nähere Informationen zu DNS allgemein erhalten Sie auf Wikipedia: Domain Name System.

  • Nennen Sie Idenos (oder bei einem eigenen Server Ihrem Systemadministrator) die verwendeten Domains und Subdomains, da diese zusätzlich auch noch in der Konfiguration des Apache-Webservers eingetragen werden müssen
  • Bis die Domain aufrufbar ist kann es bis zu 24 Stunden dauern. Um zu prüfen, ob die Umstellung erfolgreich abgeschlossen wurde geben Sie in der Windows-Eingabeaufforderung den Befehl "ping www.meine-domain.de" ein. Dann müssten die neue IP erscheinen (85.88.2.186)


Wichtig
Sobald die Domain zugewiesen wurde, können auch externe Besucher auf die Seite zugreifen. Setzen Sie die Agentur deshalb im Backend → Einstellungen → Konfiguration → Kunden auf Zugang für Backendbenutzer. Dann können nur noch die Benutzer mit den Zugriffsrechten auf das Backend sich einloggen und auf das System zugreifen.

Suchmaschinenoptimierung

Wenn Sie wünschen, dass Ihre Agentur von den Suchmaschinen erfasst wird, sollten Sie von einer den Suchmaschinen bereits bekannten Seite einen Link auf Ihre neue Domain setzen. Optimalerweise sollte diese Seite bereits länger online sein (Pagerank) und nach Möglichkeit auf einem anderen Server gehostet werden. Eine Option dafür sind z. B. Linkpartnerschaften. Alternativ können Sie die neue Domain auch bei jeder Suchmaschinen einzeln melden (http://www.google.de/submityourcontent/tools.html#web) oder versuchen in manuell gepflegten Web-Verzeichnissen einen Eintrag zu erhalten (http://www.dmoz.org).

Hinweis
Zur Optimierung für Suchmaschinen sollten Sie es vermeiden den gleichen Inhalt unter verschiedenen Domains anzubieten. Deswegen sollten Sie nur eine Hauptdomain direkt auf dem Server verweisen und die weiteren Domains (auch ohne www.) direkt auf die Hauptdomain weiterleiten.


E-Mail-Konfiguration

Auf unseren Servern können keine E-Mail-Konten eingerichtet werden. Es besteht aber grundsätzlich die Möglichkeit über einen speziellen DNS-Eintrag die E-Mail-Konten auf einen anderen Server einzubinden.

Ergänzend zur obigen Zone, können E-Mail-Server wie in folgendem Beispiel definiert werden: 

mail.domain.tld.   IN    A       231.123.1.23
domain.tld.        IN    MX      1 mail.domain.tld
domain.tld.        IN    TXT     "v=spf1 a mx ~all"

Weitere Informationen erhalten Sie bei Ihrem Domainprovider oder Ihrem Systemadministrator.

Sub-Domain

Verwenden Sie unsere Hosting-Dienstleistung in Verbindung mit xmstore erhalten Sie von uns nach der Installation eine kostenfreie Subdomain (name.xmstore.de) die Sie für die Einrichtung des System verwenden können.
