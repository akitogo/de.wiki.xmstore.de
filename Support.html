Voraussetzungen für die verschiedenen Bereiche

xmstore ist eine umfangreiche und in einen hohen Grad individualisierbare Lösung. Zur selbständigen Anpassung und dem Betrieb des Systems, benötigen Sie technische Erfahrungen in verschiedenen Bereichen. Alternativ können Sie uns oder einen Techniker beauftragen, diese Leistungen für Sie zu erfüllen. Hier die verschiedenen, benötigten Kenntnisse die für einzelnen Bereiche.


Verwaltung des Systems

Als Inhaber oder Betreiber des Systems benötigen Sie die folgenden Kenntnisse

  • Umfangreiche Internet-Kenntnisse (Anwender)
  • Grundsätzliche kaufmännische Kenntnisse (Buchhaltung)
  • Möglichkeiten verschiedener Dateiformate
  • Hochladen von Dateien per FTPDas File Transfer Protocol (FTP) dient zum Dateiaustausch zwischen unterschiedlichen Systemen im Internet (Upload/Download).


Anpassung der Templates

Zur Anpassung des Templates benötigen Sie einen Webdesigner. Dieser muss mit folgenden Arbeiten vertraut sein:

  • Bearbeiten von Dateien per FTPDas File Transfer Protocol (FTP) dient zum Dateiaustausch zwischen unterschiedlichen Systemen im Internet (Upload/Download).
  • Webdesign mit HTMLDie Hypertext Markup Language ist die Sprache, welche am häufigsten zum Aufbau von Internetseiten verwendet wird. Sie basiert auf der universellen Struktursprache "XML"., JavascriptJavascript ist eine von Netscape entwickelte objektorientierte (an Java angelehnte) Scriptsprache und kann clientseitig im Browser ausgeführt werden um dem Besucher interaktionen mit der Website zu ermöglichen. und CSSCSS steht für Cascading Style Sheets. CSS ist eine offiziell normierte Ergänzungssprache zu HTML. Mit CSS kann man das Layout einer Webseite bestimmen. Mit Styles können HTML-Tags z.B. Texte und Hintergründe von Webseiten gestaltet und formatiert werden.
  • Konvertierung und bearbeiten von Grafiken mit einem Bildbearbeitungsprogramm


Systemadministrator des Servers

Falls Sie einen eigenen ServerBezeichnung für einen Computer, der Ressourcen verschiedenster Art anderen Computern im Internet zur Verfügung stellt. verwenden, ist es notwendig einen erfahrenen Systemadministrator für die langfristige Betreuung Ihres Servers mit einzubinden:

  • Vor- und Nachteile verschiedener Distributionen
  • Steuerung des System über die KonsoleDie Shell (dt. "Schale") ist der interaktive Kommando-Interpreter eines Unix-basierten Systems (vor allem Linux). Sie läuft üblicherweise in einer Konsole. Mit ihr lässt sich ein System textbasiert nutzen und administrieren.
  • Installation und Konfigurationen des Servers
  • Planung und Umsetzung von Datensicherungskonzepten


Achtung
Bei technischen Fragen oder Probleme im Zusammenhang mit Ihrem ServerBezeichnung für einen Computer, der Ressourcen verschiedenster Art anderen Computern im Internet zur Verfügung stellt., wenden Sie sich bitte an Ihren Systemadministrator oder Provider. Wir können Ihnen dabei leider keine Unterstützung anbieten. Wünschen Sie hier eine Betreuung durch unser Team, nutzen Sie bitte unseren HostingUnter Hosting oder Webhosting versteht man die Speicherung und Veröffentlichung von Webseiten und Daten auf dem Webserver.-Service.


Anpassung der Software

In manchen Editionen ist es möglich in den Programmablauf von xmstore technisch einzugreifen (z. B. Hookpoints). Diese Anpassungen sollten ausschließlich von erfahrenden Software-Entwickler durchgeführt werden, da bereits kleine Fehler in diesen Bereich schwere Folgen mitsichziehen können.

  • Programmiersprache PHPDer PHP Hypertext Preprocessor ist eine in HTML eingebettete Scriptsprache welche Serverseitig ausgeführt wird. Mit Hilfe von PHP kann man dynamische und interaktive Websites programmieren.
  • DatenbankEine Datenbank ist eine meist relational strukturierte Sammlung von Daten, die normalerweise in einer oder in mehreren zusammengehörenden Dateien gespeichert ist. MySQLDie relationale Datenbank MySQL (von Oracle) ist als Open-Source-Software für verschiedene Betriebssysteme verfügbar und bietet damit die Grundlage vieler Webauftritte.
Wichtig
Wünschen Sie hier Anpassungen sollten Sie diese Änderungen direkt von uns umsetzen lassen.
Achtung
Eine grundsätzliche Anpassung von xmstore außerhalb dieser ausgewiesenen Programmbereiche von einem externen Entwickler ist nicht zulässig (siehe AGB) noch möglich (Verschlüsselung, Updates). Grundsätzlich können wir eine vollkommen individuelle Software-Lösung (Enterprise) auf Basis von xmstore für Sie entwickeln.

Wiki-Handbuch

In diesem Handbuch in Form eines Wikis finden Sie ausführliche Informationen Installation, Einrichtung und Bedienung von xmstore. Das Handbuch wird durch neue Beträge regelmäßig erweitert, um möglichst vielen offenen Fragen gerecht zu werden.

xmstore-Forum

Werden offene Fragen im Handbuch nicht erläutert, besteht die Möglichkeit diese Anfrage an andere Benutzer in xmstore-Forum zu stellen. Bitte beachten Sie dabei die Forum-Regeln.

Support-Leistungen

Wenn Sie eine direkte und umfassenden Betreuung durch unser Team wünschen, bieten wir Ihnen gerne unsere Support-Leistungen an. Unser Stundensatz beläuft sich auf 90 Euro pro Stunden (kleinste abrechnenbare Einheit sind 15 Minuten 22,50 Eur). Den aktuellen Stand der Umsetzung können Sie auch im Kundenbereich verfolgen. Dort sehen Sie jederzeit alle beauftragen und die bereits durchgeführten Aufträge und haben so jederzeit einen Überblick über die Kosten.

Erstellung von technischen Umsetzungskonzepten

Bei größeren, individuellen Anforderungen bieten wir die Möglichkeit ein grobes Konzept zur Machbarkeit und zur Einschätzung des Aufwands durch unser Team zu erstellen zu lassen. Der Aufwand für das Konzept entsteht in diesem Fall unabhängig von der Beauftragung.
