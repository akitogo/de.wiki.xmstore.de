Wichtiges zum Start Ihres Systems

Hier eine komprimierte Seite mit den wichtigsten Schritten zum schnellen Start Ihres Systems. Bevor Sie die Website mit dem Angebot online ist, sollten Sie diese Punkte durchgeführt haben.

1. Design: Template

Wählen Sie eines der Templates aus und binden Sie Ihr Logo ein.

→ Hauptartikel: Templates

2. Inhalte: Seiten & Systemtexte

Leben Sie die Hauptseiten des Systems im Frontend an und passen die bei Bedarf die Texte vom System an.

→ Hauptartikel: Seiten
→ Hauptartikel: Texte und Begriffe

3. Rechtliches: Betreiberdaten, AGB & Datenschutz

Tragen Sie Ihre Betreiberdaten ein und binden Sie die Geschäftbedingungen ein.

→ Hauptartikel: Betreiberdaten
→ Hauptartikel: Rechtliches

4. Konfiguration: Systemeinstellungen

Bearbeiten Sie ein Preismodell und die Anwendungsregeln in den Konfiguration.

→ Hauptartikel: Preismodelle
→ Hauptartikel: Einstellungen#Preisebenen
→ Hauptartikel: Zahlungssysteme

5. Medien: Upload & Wasserzeichen

Legen Sie bei Bedarf Kategorien an, definieren Sie die Wasserzeichen und laden die ersten Medien in das System.

→ Hauptartikel: Kategorien
→ Hauptartikel: Einstellungen#Vorschau
→ Hauptartikel: Upload-Verzeichnis
→ Hauptartikel: Freigabe
→ Hauptartikel: Verschlagwortung

6. Start: Systemtest & Domain

Testen Sie das System durch einen Testbestellung, Tragen Sie die richtige Domain ein (falls noch nicht passiert) und verlinken Sie das System extern, damit die Suchmaschinen auf die Seite aufmerksam werden. Informieren Sie Ihrer Kunden & Partner zum Start des Systems. Starten Sie eine Datensicherung des Systems

→ Hauptartikel: Domain
→ Hauptartikel: Wartung#Datensicherung
